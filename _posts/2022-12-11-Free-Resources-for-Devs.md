# Cloud Resources

### Oracle Cloud

Oracle cloud offers a huge library of free resources for life, it includes an 4 cores ARM instance with 24 GB of RAM and 200 GB of hard drive, it also offers a 1 GB link.

[https://www.oracle.com/cloud/free/](https://www.oracle.com/cloud/free/)

### Google Cloud

Google cloud offers a 1 core tiny instance free for life witch can also bee used.

[https://cloud.google.com/free/](https://cloud.google.com/free/)

### IBM Cloud

IBM's offers a small document database, like MongoDB to store up to 1 GB data. It also offer cloud functions and object storage.

[https://www.ibm.com/cloud/free](https://www.ibm.com/cloud/free)

### Cloudflare

Cloudflare, most famous DNS provider offers an interesting object storage options for initial developers.

[https://www.cloudflare.com/en-gb/plans/free/](https://www.cloudflare.com/en-gb/plans/free/)

# Redirect

### Short.io

Short.io offers a redirect service for custom domain so you can track your small projects.
